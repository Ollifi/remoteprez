remoteprez
===

Introduction
---

Demo: http://www.youtube.com/watch?v=Sa-kmN7n92o

A Google Chrome extension to remotely control an HTML5 presentation.

Currently, the following presentations are supported:

- [Reveal.js][1]
- [impress.js][2]
- [html5slides][3]
