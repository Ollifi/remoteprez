Introduction à remoteprez

remoteprez est une extension Google Chrome permettant de contrôller à distance des présentations HTML5.

Regardez la démo!

À l'heure actuelle, les présentations suivantes sont supportées :

  • Reveal.js
  • impress.js
  • html5slides
  • csss

Téléchargement

Téléchargez l'extension officielle sur le Chrome Web Store.
